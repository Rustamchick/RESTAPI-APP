## TODO REST API (Go)

Реализован REST API для TODO-приложения с JWT-аутентификацией.

Приложение построено на стандартном пакете `net/http`, при этом Gin используется для маршрутизации, middleware и обработки HTTP-контекста.

Архитектура проекта разделена на слои с явным разделением ответственности:
- **handler** — HTTP-слой, обработка запросов и ответов
- **service** — бизнес-логика приложения
- **repository** — работа с базой данных

Зависимости между слоями внедряются через конструкторы (dependency injection), что упрощает тестирование и расширение приложения.

В качестве хранилища данных используется **PostgreSQL**. Схема базы данных управляется через миграции.  
Конфигурация приложения вынесена из кода и загружается через **Viper** (переменные окружения и конфигурационные файлы).

Проект полностью контейнеризован и запускается через **docker-compose**, который включает:
- контейнер с Go-приложением
- контейнер с PostgreSQL
- отдельный контейнер для запуска миграций базы данных

Проект ориентирован на практики production-разработки: воспроизводимое окружение, разделение конфигурации и кода, чистая архитектура.
